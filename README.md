# Hytale Mod: Crystal Anvil + Crystal Sword (Create A Crafting Bench)

Companion mod for the [Hytale Modding Manual](https://hytale-moddings.github.io/hytale-modding-docs/) tutorial on **Create A Crafting Bench**.

**Repository:** [hytale-moddings/hytale-mods-custom-bench](https://github.com/hytale-moddings/hytale-mods-custom-bench)

## What This Mod Does

Adds a **Crystal Anvil** crafting bench plus a **Crystal Sword** that can be crafted at it, **Light Charges** ammo, and a **Crystal Light** element. This is the v2.0.0 bundle that pairs with the projectile-weapons tutorial.

### Features

- Custom crafting bench (`Crystal Anvil`) with `State.Id="crafting"` for UI
- Crystal Sword recipe gated behind the bench
- Light Charges ammo for projectile attacks
- Crystal Light element for power scaling
- Multilingual: EN-US, ES, PT-BR

## Installation

1. Copy the mod folder to `%APPDATA%/Hytale/UserData/Mods/CreateACraftingBench/`
2. The folder name **must** be `CreateACraftingBench`
3. **Requires:** [CreateACustomBlock](https://github.com/hytale-moddings/hytale-mods-custom-block) and [CreateACustomTree](https://github.com/hytale-moddings/hytale-mods-custom-tree)
4. Start a Hytale server with the mod enabled
